# MichinakaMap – User Guide

Welcome to the official user guide for MichinakaMap.

## Features
- Display the midpoint between two locations
- Simple and intuitive interface
- Supports multiple map providers

## How to Use
1. Enter two locations
2. Tap “Search”
3. The midpoint will be displayed on the map
